Motion and Animation

Colliding pyramids

An automatic animation causing pyramids to move through each other. Unfortunately, A-Frame doesn't provide the pyramid primitive, so these are selfmade with triangles and a rectangle. A little exercise with animation and the particle system.

No user action required. Just enjoy.

Find an Exit

A scene to get the user more involved and require reaction. There is no collision applied. But you may want to move around trying to avoid contact with the stones.

Requires WASD or arrow keys.
